Matlab implementation of the Compact Binary Face Descriptor (CBFD) algorithm.
Please use the citation provided below if it is useful to your research:

Jiwen Lu, Venice Erin Liong, Xiuzhuang Zhou, and Jie Zhou, Learning Compact Binary Face Descriptor for Face Recognition, IEEE Transactions on Pattern Analysis and Machine Intelligence (T-PAMI), 2015, in press.

run 'demo.m' to test on FERET set using pre-learned CBFD parameters
